SAP NetWeaver Gateway Connectivity Addon for Spring Roo

Intro | Tutorial | Overview

Introduction

What is it about?

In short, it's about creating Java-based web applications which are connected to NetWeaver Gateway systems - and all this very fast.

How is it done?

It's done by creating the web application using the Rapid Application Development (RAD) tool Spring Roo, together with the Roo addon provided by this project to configure connectivity to specified NetWeaver Gateway systems as well as accessing the exposed entity sets. The timespan between starting with an empty directory and ending with a running web application should be just few minutes.

There is also the option to use the SAP HANA Cloud Addon for Spring Roo in order to deploy the application on SAP HANA Cloud.

What is all this stuff?

Spring Roo is a console-based Rapid Application Development tool maintained by SpringSource. Roo generates 100% pure Java code, managed as a standard Maven project that can easily be imported into an IDE like Eclipse. Roo can be extended by addons, that provide additional functionality to the tool.

The SAP NetWeaver Gateway Connectivity Addon for Spring Roo is an addon for Roo which adds commands that enable connectivity to NetWeaver Gateway services. Moreover, by providing the necessary methods for CRUD operations, the web application could be used as a service interface.

Try it!

If you want to give it a try, just download the addon package and follow the tutorial we prepared. The tutorial will show you how to create a web application using the provided addon in just a few minutes.
