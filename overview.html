SAP NetWeaver Gateway Connectivity Addon for Spring Roo

Intro | Tutorial | Overview

What is it about?

It's about creating Java-based web applications which are connected to NetWeaver Gateway systems - and all this very fast.

What technology is used?

Spring Roo is used as a Rapid Application Development (RAD) tool, which creates web applications consisting of 100% pure Java code, managed as s standard Maven project.

What kind of results do I get?

Using the addon provided by this project, Roo generates a web application containing (some) entities whose persistency is translated to NetWeaver Gateway services and a frontend providing a web UI for interfacing all the CRUD operations. You may use this as a starting point for a full blown web application, or leave it as-is and use it e.g. as cloud backend for a mobile app that connects to the automatically generated REST interfaces of the backend.

What do you provide?

Software

  • Roo addon to configure projects for NetWeaver Gateway connectivity
  • ZIP package of plugin/addon above (precompiled binaries) + installation instructions

Instructions

What additional commands the addon provides?

gateway setup
gateway define odata_endpoint --Name --URL --USER --PASSWORD --CSRF_MODE --HTTP_PROXYHOST --HTTP_PROXYPORT --TIMEOUT_CALL
gateway entity --namespace --remoteEntitySet --import_all
gateway field --entityClass --fieldName
